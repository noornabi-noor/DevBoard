Question-1: What are the different ways to select an element in the DOM?

We can select elements in the DOM using various methods like: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Within an element, retrives the HTML content including tags
innerText: return only visible HTML content
textContent: Within an element, retrives all text content.

Question-3: What is event delegation in the DOM?

It is a JS pattern which adding event to multiple child.

Question-4: What is event bubbling in the DOM?

It is a technique where an event starts from the target element.

Question-5: How do you create, add, and remove elements using JavaScript?

Create Element: Use document.createElement(tagName) .
Add Element: Using appendChild() methods Like: document.body.appendChild(newElement);
Remove Element: Use methods remove() . Like newElement.remove();
